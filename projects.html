Anubis

This is a web app that allows users to add tv shows/movies to their watchlist to keep track of everything they want to watch
The app uses the API from TheMovieDB to fetch trending movies and shows and also facilitates the search features

Flack

This is a web app that uses SocketIO to asynchronously communicate with the back end so the page never reloads after a form is submitted
Users can create channels and chat with other users that enter the channels, similar in spirit to Slack

Flask | Javascript | Python

Astron

This web app fetches the Astronomy Picture of the Day from the NASA API for any date entered
Some information about the photo is also provided

Flask | Python

Nations

This simple web app fetches various bits of information about any country that the user searches for
